# PetShop Ecommerce

Um sistema completo de e-commerce para PetShop, desenvolvido com as tecnologias mais modernas do mercado.

## Tecnologias Utilizadas

- **Fron-end**: React 18
- **Build Tool**: Vite
- **Estilização**: Tailwind CSS & shadcn/ui
- **Linguagem**: TypeScript
- **Gerenciamento de Estado**: Zustand / TanStack Query (se aplicável)
- **Animações**: Framer Motion

## Como executar o projeto localmente

1. Clone o repositório
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```
4. Acesse `http://localhost:8080` no seu navegador.

## Scripts Disponíveis

- `npm run dev`: Inicia o servidor de desenvolvimento.
- `npm run build`: Gera a versão de produção do projeto.
- `npm run lint`: Executa a verificação de código (ESLint).
- `npm run preview`: Visualiza a versão de produção localmente.
- `npm run test`: Executa os testes unitários.
